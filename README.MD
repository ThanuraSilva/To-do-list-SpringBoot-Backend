## Introduction with how to use
#### This ToDo List API is developed under Spring Boot Framework and here applied Spring Boot Web Starter pack to initiate the developing environment.
#### The business logic of the API is to save the ToDo task with a dedicated User. It includes all the basic CRUD operations regards the business logic. This initial attempt to get hands-on skills about Spring Boot Framework. 
#### There are further updated versions that will be released for this small project.
#### Here are the two releases of the API:
#### V 1.0.0 - Public release with includes MySQL as the DMBS - You can grab it from the main branch ![MySQL](https://img.shields.io/badge/mysql-%2300f.svg?style=for-the-badge&logo=mysql&logoColor=white)

#### V 1.1.0 - Public release with include feature update with MongoDB as the DMBS - You can grab it from the feat/mongoDB-impl branch ![MongoDB](https://img.shields.io/badge/MongoDB-%234ea94b.svg?style=for-the-badge&logo=mongodb&logoColor=white)

#### Its experienced both SQL and NoSQL DBMS applications in a single API.
